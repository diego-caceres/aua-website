## Design Context

### Project
**AUA — Asociación Uruguaya de Apneistas**
Official AIDA affiliate in Uruguay for freediving (apnea). A non-profit sports association that promotes recreational and competitive freediving. Content is in Spanish.

### Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS (custom config)
- Fonts: **Cormorant Garamond** (headings, `font-heading`) + **DM Sans** (body, `font-sans`)
- Icons: lucide-react
- Routing: React Router DOM
- No component library

### Color Palette
- Primary blues: `blue-500` (#3b82f6) → `blue-950` (#172554)
- Accent: `blue-300` / `blue-400` for highlights
- Background: animated scroll-driven gradient from `blue-500` to `blue-900`
- Text: white on all backgrounds
- Surfaces: `bg-white/5` (subtle container), glassmorphism reserved for image cards only

### Existing Patterns
- Animated fish SVGs in fixed background (opacity 15–30%) — the site's most memorable element
- Scroll-based depth transition: background darkens as user scrolls (light blue → deep blue)
- Button hierarchy: primary (`bg-blue-500`, filled, shadow) vs secondary (ghost, `border-white/30`)
- Section h1s: `border-l-4 border-blue-400 pl-3` — one per page only
- Other headings: `pb-2 border-b border-white/20`
- Sidebar nav on home page only (desktop)

### Users
Two equally important audiences:
1. **Curious newcomers** — people discovering freediving for the first time, evaluating whether to join
2. **Existing freedivers** — athletes looking to affiliate with AIDA Uruguay, join events, or connect with the community

Both arrive needing to quickly understand what AUA is and how to join.

### Brand Personality
**Welcoming · Curious · Adventurous**

AUA is a community first, a sports federation second. The tone is warm and open — anyone can explore freediving, regardless of level. Institutional credibility (AIDA affiliation) matters, but it should feel approachable, not bureaucratic.

### Emotional Goals
The site should make visitors feel:
1. **Welcomed** — this community is for me, I belong here
2. **Awe** — the underwater world is vast and beautiful, I want to explore it
3. **Calm** — freediving is meditative; the interface should carry that stillness

### Aesthetic Direction
- **Reference**: AIDA International visual identity — institutional, credible, clean. Use it as a baseline for trustworthiness, not as a design ceiling.
- **Tone**: Editorial and immersive. The animated underwater background is the hero; everything on top should let it breathe.
- **Theme**: Light-on-dark (blue world). Deep ocean as metaphor — depth, stillness, discovery.
- **Anti-patterns to avoid**: Neon/glowing accents, glassmorphism overuse, identical card grids, gradient text, all-caps subheadings, bouncy animations.

### Design Principles
1. **Let the ocean breathe** — The animated background is the identity. Content sits on top of it lightly, not competing with it. Use restraint with backgrounds and blur effects.
2. **One voice, two audiences** — Every page should work for both a first-time visitor and a returning member. Lead with welcome, follow with depth.
3. **Hierarchy by restraint** — Limit accent treatments (border-left, color, shadow) to one instance per context. When everything is accented, nothing is.
4. **Calm over urgency** — Avoid aggressive CTAs, timed prompts, or high-pressure patterns. The sport values patience and stillness; the interface should too.
5. **Institutional warmth** — AIDA credibility matters, but always filtered through community tone. Rules and certifications are safety, not gatekeeping.
